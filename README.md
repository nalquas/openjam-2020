## Controls
### Keyboard
- Turn ship: Left/Right or A/D
- Thrust: Up or W
- Boost: Shift
- Shoot: Space
- Menu: ESC
- Toggle fullscreen: F11

### Gamepad (assuming Xbox layout)
- Steer ship: Left stick (or turn using directional pad)
- Thrust: A
- Boost: B
- Shoot: X
- Menu: Start
- Toggle fullscreen: Select

### Mouse
- Steer ship: Mouse position
- Thrust: Left button
- Boost: Middle button
- Shoot: Right button
